# Wireframes à produire visuellement

1. Accueil : hero, CTA, modes de transaction, produits récents.
2. Catalogue : filtres à gauche/haut, grille produits, tri.
3. Produit : image, description, prix, actions achat/encherir/négocier.
4. Connexion / inscription : formulaires centrés.
5. Panier : liste produits, quantités, total.
6. Paiement : résumé, adresse, méthode fictive.
7. Espace client : cartes statistiques.
8. Espace vendeur : annonces, ventes, création.
9. Enchère : prix actuel, timer, historique.
10. Négociation : fil de messages et actions.
11. Admin : statistiques, modération, utilisateurs.

Les wireframes tiennent compte de la présence d’une bannière visuelle, de cartes catégories illustrées et de cartes produits avec images.
