# Storyboard principal

Visiteur → Accueil → Catalogue → Détail produit → Connexion / Inscription.
Acheteur → Connexion → Catalogue → Produit → Panier → Paiement simulé → Confirmation → Notification.
Acheteur enchère → Produit en enchère → Offre → Historique → Fin enchère → Notification gagnée/perdue.
Acheteur négociation → Produit négociable → Offre → Contre-offres → Acceptation/refus/abandon.
Vendeur → Espace vendeur → Créer annonce → Modération admin → Publication → Vente/négociation/enchère.
Admin → Espace admin → Modérer annonces → Gérer utilisateurs → Journal d’actions.
